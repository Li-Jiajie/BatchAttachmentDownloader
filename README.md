# BatchAttachmentDownloader

邮件附件批量下载 v1.3.1

用于批量下载邮件中的附件，支持IMAP与POP3协议，支持筛选，具有多种保存模式。


---
v1.3.2 更新 2021.04.23

去除dateutil包依赖

内部时间改为时间戳方式存储

---
v1.3.1 更新 2021.04.21

支持从X-QQ-mid字段中读取邮件时间，可以解决在QQ邮箱下部分邮件时间解析失败的问题

---
v1.3.0 更新 2020.10.22

加入支持IMAP协议

解决部分邮件头无Data字段的问题
